Numeric Pipes of angular - usually included in the commom module

{{totalSells +1234}} = {{totalSells|number}} (using number pipe)
{{totalSells}} = {{totalSells|number:'1.2-2'}} (pipe with config only show 2 decimals)
{{totalSells}} = {{totalSells|currency:'BOB'}} (using currency pipe)
{{totalSells}} = {{totalSells|currency:'':'symbol-narrow'}} (currency pype w/ symbol-narrow)
{{totalSells}} = {{totalSells|currency:'':'symbol-narrow':'1.6-6'}} (currency pype w/ symbol-narrow and decimal definition)
{{percent}} = {{percent|percent:'1.2-2'}} (using percent pipe)
